Web Storage

Web storage, sometimes known as DOM storage (Document Object Model storage), provides web apps with methods and protocols for storing client-side data.

Web storage offers two different storage areas -- local storage and session storage. We use session storage on our website. We store only some essential info such as authentication token.

You can continue accessing our public services even after denying the storage permission.
